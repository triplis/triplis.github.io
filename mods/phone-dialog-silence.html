Block Phone Dialog While Silenced

Summary

Normally, the "silence" feature on the phone will block the ringing when you get a phone notification, but not much else. This mod blocks the notifications entirely when the phone is silenced.

Features

  • True Silence
  • Usage Warnings

True Silence

"Hi, I heard you got married! Click on me!"

"Hi, I heard you got married too! What a hoot!"

"I, too, heard that you got hitched. When will it ever end?"

"I heard you aren't a nice person."

"I haven't heard anything, I'm just lonely."

Your phone should now be truly silent when silenced, with the dialog notifications being blocked entirely. I don't recommend using this mod if you want to be able to interact with the dialog notifications for some reason when your phone is silenced, as it doesn't let them show at all when the phone is silenced.

^

Usage Warnings

It's unlikely this mod would break anything important. However, it could change your experience in more ways than just having the random dialog notifications blocked when your phone is silenced. For example, I noticed in my testing that it will block the pop up notification for your boss calling you if you're late for work (emphasis on IF your phone is silenced). I doubt this sort of thing is going to bother the type of person who wants to be able to silence the dialog spam, but I figured I should make you aware before you start using it.

^
